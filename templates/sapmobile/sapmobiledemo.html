SAP Mobile Applications based on HTML5

The prototype applications below result from a project with the goal of developing mobile applications for SAP using business server pages on the SAP server together with HTML5 mobile toolkit jQueryMobile. This combination proved to be a very good solution for fast development of SAP business apps that work in most mobile devices.

The first demo is for an app that monitors and updates the status of shipment documents.

To see the demo open the url labs.cognitiva.com/sapmobile in a mobile device, or use the QR code below to open that page.
